palasimi

noun [pa.laˈsi.mi]

  1. A collection of colexification graphs extracted from Wiktionary. Colexification occurs when a language uses the same word for different concepts. The website provides an interactive visualization of common colexifications found in different languages.

Etymology

pala + simi

From Spanish palabra ("word") and similar ("similar").

Data

All the data used by the website is derived from Wiktionary. The copyright of the original work belongs to Wiktionary's editors and contributors.
